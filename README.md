# Fathom

A 2D underwater adventure game where you dive deep, catch fish, and upgrade your equipment to explore the ocean depths.

## Game Concept

Control a diver exploring underwater environments, managing oxygen while hunting fish for profit. Use your earnings to purchase equipment upgrades that allow you to dive deeper and stay underwater longer.
